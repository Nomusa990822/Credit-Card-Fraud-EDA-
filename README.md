# Credit Card Fraud Detection — Exploratory Data Analysis

## Overview

This project performs exploratory data analysis on a credit card transaction dataset to identify patterns associated with fraud.

The analysis focuses on understanding transaction behavior, class imbalance, and key indicators of fraudulent activity.

## Dataset

Dataset available on Kaggle:
https://www.kaggle.com/datasets/kaushalnandania/credit-card-fraud-detection

## Key Analysis Areas

- Fraud vs non-fraud distribution
- Transaction amount analysis
- Fraud by category
- Fraud by gender and state
- Time-based fraud patterns
- Age and demographic insights
- Correlation analysis

## Key Findings

- Fraud is rare but significant (~0.52%)
- Fraudulent transactions tend to have higher amounts
- Certain categories show higher fraud rates
- Fraud is more common during late-night hours
- Time features are strong indicators of fraud behavior

## Tools Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Kaggle Notebooks

## How to Run

Run the notebook in Kaggle or locally using Jupyter Notebook.
